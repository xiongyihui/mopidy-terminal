Mopdiy extension for web terminal


Installation
============

Install by running::

    pip install Mopidy-Terminal

Or, if available, install the Debian/Ubuntu package from `apt.mopidy.com
<http://apt.mopidy.com/>`_.


Configuration
=============

Mopidy-Terminal to your Mopidy configuration file::
